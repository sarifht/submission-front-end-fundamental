### Tutorial cara menjalankan project ini:

1. Install semua package yang berada di package.json

   ```
   npm i
   ```

2. Untuk menjalankan project-nya

   ```
   npm run start-dev
   ```

3. Untuk menjalankan prettier

   ```
   npx prettier . --write
   ```
